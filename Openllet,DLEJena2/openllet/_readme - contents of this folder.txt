The openllet examples here can be compiled and run using:
_run_ApacheMaven_openllet2.6.3_mvnrepository_MultipleJars.bat
_run_ApacheMaven_openllet2.6.3_mvnrepository_SingleJar.bat
_run_ApacheMavenMultipleJars.bat
_run_ApacheMavenSingleJar.bat
_run_manualJarChoice.bat

Each "_run_..." batchfile writes its outputs to the respective output_... folder
and uses the respective lib folder.

The source code is in:
The openllet 2.6.2 examples: openllet\examples
The openllet 2.6.3 examples that have changed from V2.6.2 openllet\examples263
The examples that were modified by Jevan Pipitone openllet\examplesModifiedByJevan

The data is in:
The openllet 2.6.2 data: data
The openllet 2.6.3 data that has changed from V2.6.2: data263
The data that is used by Jevan Pipitone's examples: dataFromJevan
